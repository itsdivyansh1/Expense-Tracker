# Expense Tracker App

### Technologies used
- [Python](https://www.python.org/downloads/)
- [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)

### Dependencies
- Install Tkinter
```text
pip install tkinter
```

- Install Custom Tkinter
```text
pip install customtkinter
```

- Install pillow
```text
pip install pillow
```
